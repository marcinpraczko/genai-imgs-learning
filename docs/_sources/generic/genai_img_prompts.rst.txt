Prompts
=======

During working with Gen AI ``Prompts`` are very important, master prompts leads to having much better expected
results. Having some default prompts allows have good fun what ``AI`` is able to generate, however knowing
how to write prompts leads to getting more accurate results.

Details about prompts
---------------------

- Be specific - especially with objects - there is difference between ``ship`` and ``sailing ship``
- Write about background - ``clouds, moon, night, day, clear sky``
- Write about colors - ``silver knight``, ``red knight``
- Write about action - ``running``, ``jumping``, ``sitting``
- Write about expected results - ``High realistic Image`` and ``concept art```, or ``cartoon``.
